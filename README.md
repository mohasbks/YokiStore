## ✨ Premium Features

- **🛍️ Seamless Shopping:** Intuitive product browsing, filtering, and a fully functional cart system.
- **🔍 Smart Search:** Real-time product filtering using Vue watchers (Search by title, description, category).
- **🛒 Dynamic Cart:** Add/remove items with animated feedback, quantity controls, and real-time total calculation.
- **💾 LocalStorage Sync:** Cart state persists across browser sessions automatically.
- **🎯 API Integration:** Live product fetching from Fake Store API with loading states and error handling.
- **📱 Responsive UI:** Built with Bootstrap 5 grid system for a perfect shopping experience on desktop and mobile.

## 🚀 Quick Start

Follow these steps to run the project locally:

**1. Clone the repository:**
```bash
git clone https://github.com/mohasbks/YokiStore.git
cd YokiStore
```

**2. Install dependencies:**
```bash
npm install
```

**3. Run the development server:**
```bash
npm run dev
```

**4. Build for production:**
```bash
npm run build
npm run preview
```

## 🌐 Deployment

This project is optimized and configured for instant deployment on **Vercel** with a custom `vercel.json` file handling build settings and routing configuration.

1. Fork this repository.
2. Sign in to Vercel and import the forked repo.
3. Vercel will automatically detect the **Vite** configuration and handle the rest!
